# Chrome Browser


## Keys

* help: F1
* close tab: ctrl-w
* Jump to the next open tab: Ctrl + Tab or Ctrl + PgDn
* caret browsing: F7
* Open a new window in Incognito mode: Ctrl + Shift + n
