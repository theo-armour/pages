## Prompts

* Accept inline code suggestion — Tab
* Dismiss inline code suggestion — Esc
* Show next suggestion — Alt + ]
* Show previous suggestion — Alt + [
* Trigger suggestion — Alt + \
* Open ten suggestions in a separate pane — Ctrl + Enter
